# 📚 Ejemplos Básicos de Jabline

Este directorio contiene ejemplos que cubren las características fundamentales del lenguaje Jabline. Son perfectos para comenzar a aprender el lenguaje.

## 📋 Lista de Ejemplos

### 01. Variables y Operadores
**Archivo:** `01_variables_operadores.jb`
- Declaración de variables con `let`
- Operadores aritméticos (+, -, *, /, %)
- Operadores de comparación (==, !=, <, >, <=, >=)
- Operadores lógicos (&&, ||, !)

### 02. Condicionales y Bucles
**Archivo:** `02_condicionales_bucles.jb`
- Estructuras `if`, `else if`, `else`
- Bucles `while` y `for`
- Control de flujo con `break` y `continue`

### 03. Funciones
**Archivo:** `03_funciones.jb`
- Definición de funciones con `fn`
- Parámetros y valores de retorno
- Recursión
- Scope de variables

### 04. Estructuras
**Archivo:** `04_estructuras.jb`
- Definición de estructuras con `struct`
- Creación de instancias
- Acceso y modificación de campos
- Valores por defecto

### 05. Arrays
**Archivo:** `05_arrays.jb`
- Creación y manipulación de arrays
- Acceso por índice
- Bucles `for-in` para arrays
- Arrays multidimensionales

### 06. Hash Maps
**Archivo:** `06_hashmaps.jb`
- Creación de diccionarios
- Claves de diferentes tipos
- Acceso y modificación de valores
- Iteración sobre hash maps

### 07. Strings
**Archivo:** `07_strings.jb`
- Manipulación de cadenas de texto
- Concatenación
- Caracteres de escape
- Funciones built-in para strings

### 08. Números Flotantes
**Archivo:** `08_numeros_flotantes.jb`
- Trabajar con números decimales
- Operaciones aritméticas con flotantes
- Conversiones entre tipos

### 09. Constantes
**Archivo:** `09_constantes.jb`
- Declaración con `const`
- Inmutabilidad de constantes
- Diferencias con variables

## 🚀 Cómo Ejecutar

Para ejecutar cualquier ejemplo, usa el siguiente comando desde el directorio raíz del proyecto:

```bash
./jabline run examples/basic/[nombre_archivo].jb
```

Por ejemplo:
```bash
# Ejecutar el ejemplo de variables
./jabline run examples/basic/01_variables_operadores.jb

# Ejecutar el ejemplo de funciones
./jabline run examples/basic/03_funciones.jb
```

## 📖 Orden Recomendado

Se recomienda seguir los ejemplos en orden numérico, ya que cada uno introduce conceptos que se construyen sobre los anteriores:

1. **Variables y Operadores** - Fundamentos básicos
2. **Condicionales y Bucles** - Control de flujo
3. **Funciones** - Modularización de código
4. **Estructuras** - Tipos de datos personalizados
5. **Arrays** - Colecciones de datos
6. **Hash Maps** - Diccionarios y mapas
7. **Strings** - Manipulación de texto
8. **Números Flotantes** - Trabajar con decimales
9. **Constantes** - Valores inmutables

## 💡 Consejos

- Cada ejemplo incluye comentarios detallados
- Experimenta modificando los valores y viendo los resultados
- Los ejemplos están diseñados para ser independientes
- Si encuentras errores, revisa la sintaxis y los tipos de datos
