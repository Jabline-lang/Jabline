# 🚀 Ejemplos Avanzados de Jabline

Este directorio contiene ejemplos que demuestran características avanzadas y patrones de programación más complejos en Jabline. Estos ejemplos asumen familiaridad con los conceptos básicos del lenguaje.

## 📋 Lista de Ejemplos

### 01. Ejemplo Completo
**Archivo:** `01_ejemplo_completo.jb`
- Sistema completo de gestión de biblioteca
- Integración de múltiples características del lenguaje
- Estructuras complejas y manipulación de datos
- Validaciones y manejo de errores
- Caso de uso real y práctico

### 02. Operadores de Asignación
**Archivo:** `02_operadores_asignacion.jb`
- Operadores compuestos (`+=`, `-=`, `*=`, `/=`)
- Incremento y decremento (`++`, `--`)
- Optimización de operaciones comunes
- Patrones de uso eficientes

### 03. Control de Flujo Avanzado
**Archivo:** `03_control_flujo.jb`
- Bucles anidados complejos
- Uso estratégico de `break` y `continue`
- Patrones de control de flujo
- Optimización de algoritmos

### 04. Switch/Case
**Archivo:** `04_switch_case.jb`
- Declaraciones `switch` con múltiples casos
- Comportamiento de fall-through
- Casos por defecto
- Comparación con estructuras if/else

### 05. Manejo de Excepciones
**Archivo:** `05_excepciones.jb`
- Bloques `try`/`catch`/`throw`
- Propagación de errores
- Manejo robusto de excepciones
- Patrones de recuperación de errores
- Validación y sanitización de datos

## 🎯 Conceptos Cubiertos

### Patrones de Programación
- **Validación de datos**: Técnicas para verificar entrada
- **Manejo de errores**: Estrategias de recuperación robustas
- **Algoritmos complejos**: Implementaciones eficientes
- **Estructuras de datos**: Uso avanzado de arrays y hash maps

### Técnicas Avanzadas
- **Recursión profunda**: Algoritmos recursivos optimizados
- **Manipulación compleja de datos**: Transformaciones y filtrados
- **Patrones de diseño**: Implementación de patrones comunes
- **Optimización**: Técnicas para mejorar rendimiento

## 🚀 Cómo Ejecutar

Para ejecutar cualquier ejemplo avanzado:

```bash
./jabline run examples/advanced/[nombre_archivo].jb
```

Ejemplos específicos:
```bash
# Sistema completo de biblioteca
./jabline run examples/advanced/01_ejemplo_completo.jb

# Operadores de asignación
./jabline run examples/advanced/02_operadores_asignacion.jb

# Manejo de excepciones
./jabline run examples/advanced/05_excepciones.jb
```
